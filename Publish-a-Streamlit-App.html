Get started with Streamlit App

Overviewâ€‹

This article will show you how to publish a Python App with Streamlit in Katonic Platform.

In this tutorial you will:

  • spin a workspace environment with the necessary dependencies to publish a Streamlit App
  • setup and create sample Streamlit app
  • publish an App to the Katonic Deploy Launchpad

Youâ€™ll be working with sample-streamlit app.

It will take approximately 5 minutes to get this example running in Katonic platform

Setup the Workspaceâ€‹

The first step is to create a VS code Workspace capable of running your App.

  1. From the platform , click workspace.

  1. From the Workspaces Overview, click Create Workspace.

Fill the following details there.

Name : fill your workspace name

Select Environment: select vs code environment

Image: select VScode webapps image

Webapp: select Streamlit

Additional Port: 8050

Resources: Desired compute resource for workspace

  1. You will be directed to the Workspace overview window, where you can see your workspace under processing.

  1. It will take approximately 2-3 minutes to get this workspace ready.

![Untitled]publish-streamlit-app/Untitled4.png)

  1. You can connect to your workspace ,when the processing tab changes to running.

Note: Refresh the page, if your are not seeing processing to running status change after 2-3 minutes.

Click on the connect to use this Workspace for App building.

Required Git instructions and setup.â€‹

Once you connect to the your workspace , you will be taken to Vscode workspace.

Before starting anything it is advised that you should connect your github repo to the VScode workspace

first create an empty repo in your git account

clone that repo to your VScode workspace

paste below code into terminal

git clone https://github.com/account-name/repo-name.git

Cancel the pop up option.

fill your username and password/token as instructed in vscode workspace.

After logging with your git account , you will see your repo folder in workspace.

start the app development.â€‹

write your sample app.py code there or copy below sample code.

from collections import namedtuple
import altair as alt
import math
import pandas as pd
import streamlit as st
import os
routes = os.environ["ROUTE"]
"""
# Welcome to Streamlit!
Edit `/streamlit_app.py` to customize this app to your heart's desire :heart:
If you have any questions, checkout our [documentation](https://docs.streamlit.io) and [community
forums](https://discuss.streamlit.io).
In the meantime, below is an example of what you can do with just a few lines of code:
"""
with st.echo(code_location='below'):
    total_points = st.slider("Number of points in spiral", 1, 5000, 2000)
    num_turns = st.slider("Number of turns in spiral", 1, 100, 9)
    Point = namedtuple('Point', 'x y')
    data = []
    points_per_turn = total_points / num_turns
    for curr_point_num in range(total_points):
        curr_turn, i = divmod(curr_point_num, points_per_turn)
        angle = (curr_turn + 1) * 2 * math.pi * i / points_per_turn
        radius = curr_point_num / total_points
        x = radius * math.cos(angle)
        y = radius * math.sin(angle)
        data.append(Point(x, y))
    st.altair_chart(alt.Chart(pd.DataFrame(data), height=500, width=500)
        .mark_circle(color='#0068c9', opacity=0.5)
        .encode(x='x:Q', y='y:Q'))
ðŸ’¡ Note: Always define the routes as given in sample code,

routes = os.environ["ROUTE"]

when defining app.

ðŸ’¡ Always keep the port = 8050

Test your appâ€‹

Create the requirements.txt file with all required dependency.

Here is the requirements.txt code used in above app.

altair
pandas
streamlit

type following command in the terminal

pip install -r requirements.txt
streamlit run app.py --server.port=8050 --server.address=0.0.0.0

ðŸ’¡ Note: keep the path relative to your folder structure.
ðŸ’¡ Note: keep your app name as **app.py** and requirement file as **requirements.txt**

go back to the workspace

click on the Live app to see your app

your app will open on the new browser tab

Finally commit and push your code to the github.

git config user.email 'youremail@domain.com'
git config user.name 'youraccount-name'
git add .
git commit -m ' your msg'
git push
ðŸ’¡ Follow the necessary git instruction as pop up by Vscode to complete code push

Deploy your Appâ€‹

To deploy your app, go to the Deploy tab and click on the +create Deployment and select Apps

you will see below form window

fill the details as follow:

App Name: your app name

Select Environment: your app environment

Github Token: your github token

Username: your github account

Account Type: select your account personal or organisation

Repository Branch: choose app repo from drop down

Main file path: appfile name with extention

Python Version: choose desired python version for app

Resources: choose desired compute resources

Click on the Deploy button , it will take you to the Deploy window again

Hera you can see your App deployment status processing.

After few minutes status will change to running

Click on App Url to see the deployed app

You can share the App Url to showcase your work.â€‹
