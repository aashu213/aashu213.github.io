File Manager

File Manager The Console Object Browser section displays all buckets and objects to which the authenticated user has access. Use the Search bar to search for specific buckets or objects. Select the row for the bucket or object to browse. Selecting an object provides information on that object, including the option to download or delete that object. Selecting a bucket provides the option to upload new objects to the bucket. Get Started with File manager:

  1. From the platform, click File Manager.
  2. From the File Manager Overview there will be two storage buckets available, Private and Public Bucket.
  3. You can Browse the buckets by clicking on the browse button
  4. Once you click the browse button you will get redirected inside the bucket showing all the file, folder and options inside.
  5. By clicking on the folder icon, you can create a folder or a new path.
  6. By clicking on the Upload icon, you can Upload any file inside the bucket or the folder present inside the bucket.
  7. By clicking the rewind icon, you can check the file history and you can see the file after it has been deleted.
  8. By clicking the refresh button, you will refresh the File Manager overview or Bucket page.
  9. By clicking the settings Icon, you will open the settings page. From there you will get all the bucket settings related to Admin buckets and Service Accounts.
